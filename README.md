# 🐱 萌新防御 (Hajimi Defense)

> 一款可爱的 Kawaii 风格塔防游戏，保护金金枪鱼罐头免受家庭害虫的入侵！

## 🎮 游戏玩法

可爱的小猫咪守护着珍贵的金金枪鱼罐头！但是黄瓜和吸尘器正在入侵...

### 🏗️ 防御塔

| 塔 | 费用 | 能力 |
|---|---|---|
| 😸 随地吐痰的虎斑猫 | 50金币 | 快速单发攻击 |
| 😺 狙击手暹罗猫 | 75金币 | 远程高伤害 |
| 😻 扔面包的胖橘猫 | 100金币 | 范围AOE伤害 |

### 👾 敌人

| 敌人 | 生命 | 速度 | 奖励 |
|---|---|---|---|
| 🥒 黄瓜 | 30 | 快 | 10金币 |
| 🧹 吸尘器 | 100 | 慢 | 25金币 |

### 🎯 目标

- 坚守 15 波攻击
- 保护金金枪鱼罐头
- 初始 10 条生命

### 💡 提示

- 点击褐色纸箱可以打破它们，获得建造空间 (+5金币)
- 合理搭配不同类型的防御塔

## 🛠️ 技术栈

- **React 18** - 用户界面
- **TypeScript** - 类型安全
- **Vite** - 构建工具
- **Canvas 2D** - 游戏渲染

## 🚀 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 http://localhost:3000

### 构建生产版本

```bash
npm run build
```

### 预览构建

```bash
npm run preview
```
