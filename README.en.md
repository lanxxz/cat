# 🐱 Hajimi Defense (萌新防御)

> A cute Kawaii-style tower defense game! Protect the Golden Tuna Can from household pests!

## 🎮 How to Play

Cute cats are protecting the precious Golden Tuna Can! But cucumbers and vacuum cleaners are invading...

### 🏗️ Towers

| Tower | Cost | Ability |
|---|---|---|
| 😸 Spitting Tabby | 50 gold | Fast single attack |
| 😺 Siamese Sniper | 75 gold | Long range, high damage |
| 😻 Orange Bread Cat | 100 gold | AOE area damage |

### 👾 Enemies

| Enemy | Health | Speed | Reward |
|---|---|---|---|
| 🥒 Cucumber | 30 | Fast | 10 gold |
| 🧹 Vacuum | 100 | Slow | 25 gold |

### 🎯 Goal

- Survive 15 waves
- Protect the Golden Tuna Can
- Start with 10 lives

### 💡 Tips

- Click brown cardboard boxes to break them and free space (+5 gold)
- Combine different tower types strategically

## 🛠️ Tech Stack

- **React 18** - UI Framework
- **TypeScript** - Type Safety
- **Vite** - Build Tool
- **Canvas 2D** - Game Rendering

## 🚀 Quick Start

### Install Dependencies

```bash
npm install
```

### Development

```bash
npm run dev
```

Visit http://localhost:3000

### Build Production

```bash
npm run preview
```
